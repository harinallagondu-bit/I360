Challenge Summary

### Scenario
Monthly operations review of three store branches — Branch A, Branch B, and Branch C — using sales, operating cost, and customer feedback data.

---

## My Approach

### 1. Profitability Analysis
Calculated Net Profit and Profit Margin % for each branch using standard formulas:
- **Net Profit** = Total Sales − Operating Cost
- **Profit Margin %** = (Net Profit / Total Sales) × 100

| Branch | Total Sales | Op. Cost | Net Profit | Profit Margin % |
|--------|-------------|----------|------------|-----------------|
| Branch A | $45,000 | $32,000 | $13,000 | 28.9% |
| Branch B | $38,500 | $21,000 | $17,500 | 45.5% |
| Branch C | $52,000 | $48,000 | $4,000 | 7.7% |

### 2. Visualization
Created a bar chart comparing Net Profit across branches, clearly showing Branch B as the strongest performer and Branch C as the weakest despite having the highest sales volume.

### 3. Sentiment Analysis
Ranked branches based on customer feedback from best to worst experience:
1. **Branch B** — "Always has stock, friendly staff." → Fully positive on service and availability
2. **Branch A** — "Great service, but long queues." → Strong service with one fixable operational issue
3. **Branch C** — "Expensive shipping, slow delivery." → Two core pain points impacting satisfaction

### 4. Manager Pitch
- **Performance Bonus → Branch B**: Highest profit margin (45.5%), lowest operating costs, and best customer sentiment — the strongest all-round performer.
- **Process Improvement → Branch C**: Despite the highest sales ($52,000), operating costs of $48,000 leave only a 7.7% margin. Customer complaints around shipping and delivery signal urgent operational issues.

### 5. Bonus Challenge — Branch C Cost Reduction
If Branch C reduces operating costs by 15% through vendor negotiation:
- New Operating Cost: $48,000 × 0.85 = **$40,800**
- New Net Profit: $52,000 − $40,800 = **$11,200**
- New Profit Margin: $11,200 / $52,000 = **21.5%**

A 15% cost reduction would nearly triple Branch C's net profit and improve its margin from 7.7% to 21.5%.
